# Contributing Guidelines

Thank you for your interest in contributing to the **Support-Net-India** repository! Your contributions will help provide vital support to survivors of sexual abuse and domestic violence in India. To ensure the accuracy and reliability of the information, please follow these guidelines carefully.

## How to Contribute

### 1. Fork the Repository
1. Navigate to the repository.
2. Click on the "Fork" button in the upper right corner of the page.

### 2. Clone Your Fork
1. Open your terminal.
2. Clone your forked repository using the following command:
   ```sh
   git clone https://github.com/your-username/Support-Net-India.git
   ```
3. Navigate to the cloned repository:
   ```sh
   cd Support-Net-India
   ```

### 3. Create a New Branch
1. Create a new branch for your changes:
   ```sh
   git checkout -b your-branch-name
   ```

### 4. Make Your Changes
1. Add the new information or update existing content in the relevant file (e.g., `resources.csv` or `README.md`).
2. Ensure the information is accurate and up-to-date.
3. Follow the existing format and structure for consistency.

### 5. Commit Your Changes
1. Commit your changes with a clear and descriptive message:
   ```sh
   git add .
   git commit -m "Add/Update [description of your changes]"
   ```

### 6. Push to Your Fork
1. Push your changes to your forked repository:
   ```sh
   git push origin your-branch-name
   ```

### 7. Open a Pull Request
1. Go to the original repository on GitHub.
2. Click on the "New Pull Request" button.
3. Select your branch and compare it with the `unverified` branch of the original repository.
4. Provide a clear description of your changes and submit the pull request.

## Guidelines for Contributions

- **Accuracy**: Ensure that the information you provide is accurate and up-to-date. Inaccurate information can mislead survivors and potentially cause harm.
- **Relevance**: Add resources that are relevant to survivors of sexual abuse and domestic violence in India. Irrelevant information can clutter the repository and make it harder for users to find the help they need.
- **Respect Privacy**: Do not include personal or sensitive information without explicit consent. Protect the privacy and dignity of individuals and organizations.
- **Format**: Follow the existing format and structure for consistency. This ensures the repository remains organized and easy to navigate.

## Importance of Verification

Verification is a crucial step in maintaining the integrity and reliability of the **Support-Net-India** repository. Here's why it matters:

- **Accuracy and Trustworthiness**: Verified information ensures that survivors receive accurate and trustworthy support, fostering confidence in the resources provided.
- **Safety and Reliability**: Survivors rely on this repository for critical support. Verification ensures that the resources listed are safe, reliable, and genuinely available.
- **Consistency**: Verification helps maintain a consistent standard of information, making it easier for users to find and trust the resources they need.

### Verification Process

1. **Submission**: All contributions are initially submitted to the `unverified` branch via pull requests.
2. **Review**: The repository maintainers will review the changes for accuracy, relevance, and format consistency. This may involve:
   - Checking the validity of contact information.
   - Ensuring the services are free or pro bono.
   - Verifying the operational status of the listed resources.
3. **Feedback**: If any issues are found during the review, maintainers will provide feedback and request necessary revisions.
4. **Approval**: Once verified, the changes will be merged into the `main` branch, making them part of the official repository.

## Code of Conduct

By participating in this project, you agree to abide by the [Code of Conduct](CODE_OF_CONDUCT.md). This ensures a respectful and collaborative environment for all contributors.

## Contact

If you have any questions or need further assistance, feel free to open an issue or contact the repository maintainers.

Thank you for contributing to **Support-Net-India** and helping us support survivors across India!
